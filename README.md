## Rythx Editor

Rythx Editor 是一个仿照Scratch的图形化2D开发工具，基于vue开发。

## 对比Scratch

对比Scratch，Rythx拥有众多优势

### 性能优势

| 编辑器   | Rythx Editor      | Scratch                    |
| -------- | ----------------- | -------------------------- |
| 构建工具 | Vite（⚡）        | Webpack（🐢）              |
| 开发框架 | Vue               | React                      |
| 渲染引擎 | pixi.js`WebGL2⚡️` | Scratch-renderer`WebGL1🐢` |
| 桌面端   | Tauri`Rust⚡️`     | Electron `Nodejs🐢`        |

### 扩展性优势

Rythx Editor 为扩展开发者提供了更多的扩展性，以及更友好的扩展开发方式。
